Instructions (flashcard)
------------------------------------------------------------------------------------------
1. Put the TouhouDS folder inside the root folder of your flashcard.
2. Run "TouhouDS.nds"




Instructions (emulator)
------------------------------------------------------------------------------------------
You can also run the game in an emulator (no$gba recommended)

1. Use TouhouDS-EFS.nds instead of the regular TouhouDS.nds
   Beware that you can't save in the EFS version, nor is it possible to add
   characters or stages without recreating the ROM.
